# Prerequisites
* Install [NodeJS](https://nodejs.org/en/download/)
* Install [yarn](https://yarnpkg.com/getting-started/install)

## Run Dev
```shell script
yarn start
```

## Build
```shell script
yarn build
```

## Docs
```shell script
yarn styleguide
```
